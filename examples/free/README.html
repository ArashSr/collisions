FastColl

  • GIF images md5-1.gif / md5-2.gif

UniColl

  • GZip md5-1.gz / md5-2.gz

  • JPG images md5-1.jpg / md5-2.jpg

  • PDF documents md5-1.pdf / md5-2.pdf

  • PNG images

    • generic header, not 100% compatible
      • appended data md5-1.png / md5-2.png
      • appended data w/ correct CRCs md5-crc1.png / md5-crc2.png
      • synchronized comments md5-sync1.png / md5-sync2.png
    • specific header w/ common dimensions/colorspace md5-s1.png / md5-s2.png

  • MP4

    • generic header, 32b length (LTV) md5-1.mp4 / md5-2.mp4
    • generic header, 64b length (TLV) md5-l1.mp4 / md5-l2.mp4
    • specific header md5-s1.mp4 / md5-s2.mp4

  • ZIP md5-1.zip / md5-2.zip

  • PE executables md5-1.exe / md5-2.exe
  • ZIP-based formats
    • 3mf md5-1.3mf / md5-2.3mf
    • docx md5-1.docx / md5-2.docx
    • Epub md5-1.epub / md5-2.epub
    • Xps md5-1.xps / md5-2.xps
